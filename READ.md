# 💰 Finance Backend System

## 📌 Project Description
This is a Finance Backend System built using Node.js, Express, and MongoDB.  
It provides APIs for managing users, transactions, and financial summaries.

---

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Add Income & Expense Records
- View All Transactions
- Dashboard Summary (Income, Expense, Balance)

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

## ⚙️ Installation

```bash
npm install
npm run dev
